# EA Part 3 시험 직전 1페이지 압축노트

기준일: 2026-04-24

이 문서는 Part 3 직전에 `빠르게 훑는 용도`다.  
핵심은 `authority -> scope -> deadline -> form` 순서를 자동화하는 것이다.

## 1. 문제 읽는 기본 순서

- 이건 `ethics/practice`, `representation`, `collections/exam/appeals`, `filing process` 중 무엇인가
- EA가 할 수 있는가
- 대표권이 필요한가, 정보조회 권한만 필요한가
- deadline / notice stage가 무엇인가
- 어느 form / publication / document가 연결되는가

## 2. Part 3 기본 문장

- `Authority first, scope second, deadline third, form fourth.`
- `Form 2848 is not Form 8821.`
- `Collection, examination, appeals, and filing are different stages.`

## 3. Circular 230 압축

- practice before the IRS
- competence
- due diligence
- conflicts
- fees
- advertising
- written advice
- sanctions
- incompetence / disreputable conduct

함정:

- conflict와 competence를 섞음
- due diligence와 general ethics를 섞음
- sanction structure를 대충 외움

## 4. Representation 압축

- `2848` -> representation
- `8821` -> information authorization
- `CAF`
- transcripts / e-services
- supporting docs
- authoritative vs non-authoritative sources
- statutes / deadlines
- FOIA
- TAS
- identity theft

함정:

- `8821`로 representation 가능하다고 착각
- legal authority hierarchy 무시
- TAS / FOIA / identity theft를 같은 이슈로 뭉개기

## 5. Collections 압축

- IA
- OIC
- CDP / CAP
- lien / levy
- CNC
- collection statute
- TFRP
- passport revocation

폼 레이더:

- `12153`
- `9465`
- `656`
- `843`
- `1040-X`

## 6. Exams and Appeals 압축

- RAR
- 30-day letter
- CP2000
- burden of proof
- privilege
- appeals request
- conference
- settlement
- 90-day letter

함정:

- RAR / CP2000 / 30-day / 90-day를 섞음
- appeals와 exam stage를 섞음

## 7. Filing Process 압축

- accuracy
- software reliance
- duplicate entries
- record retention
- data security
- e-file provider rules
- EFIN / ERO
- `8948`
- `8879`
- `8453`
- rejected return resolution

함정:

- e-file과 representation 문제를 섞음
- rejected return 후 timely paper filing 규칙 놓침

## 8. 폼 레이더

- `2848` -> representation authority
- `8821` -> tax info authorization
- `8867` -> due diligence / preparer records
- `12153` -> CDP / collection due process
- `9465` -> installment agreement
- `656` -> offer in compromise
- `1040-X` -> amended return / refund path
- `843` -> claim / abatement path
- `8948` -> e-file exception
- `8879` -> e-file signature authorization
- `8453` -> supporting docs / e-file paper transmittal

## 9. 시험장에서 마지막으로 볼 체크 15개

- 대표 가능한가
- `2848`가 필요한가 `8821`로 충분한가
- collection / exam / appeals / filing 중 어느 단계인가
- deadline이 무엇인가
- Circular 230 issue인가
- due diligence issue인가
- conflict issue인가
- sanctions issue인가
- authoritative source를 묻는가
- lien / levy / CDP / CAP를 구분했는가
- RAR / CP2000 / 30-day / 90-day를 구분했는가
- `1040-X`와 `843`를 구분했는가
- `8948 / 8879 / 8453`를 구분했는가
- rejected return 후 후속 조치를 떠올렸는가
- form number를 보면 topic이 바로 떠오르는가

## 10. 시험 직전 암기 문장

- `2848 authorizes representation; 8821 does not.`
- `Collection stage, exam stage, and appeals stage are not the same.`
- `Circular 230 questions often turn on competence, diligence, conflict, or sanctions.`
- `Ask which notice or form controls the next procedural step.`
- `In e-file questions, think ERO, EFIN, 8948, 8879, 8453, and rejected-return timing.`
