## 최소 레이더

- `Schedule C` -> sole prop / business income and expenses
- `1065` -> partnership return
- `1120` -> C corp
- `1120S` -> S corp
- `2553` -> S election
- `4562` -> depreciation cluster
- `4797` -> sale of business property
- `3115` -> accounting method change
- `1041` -> trust / estate income tax
- `990 series` -> exempt org annual filing
- `1023 / 1024` -> exempt status application
- `4835` -> farm rental

## 오늘 산출물

- `Part 2 master forms map`

## Day 5 체크리스트

- 폼 번호를 보면 entity와 topic이 같이 떠오르는가
- `1041 / 1065 / 1120 / 1120S`를 섞지 않는가
- `1023 / 1024 / 990 series`를 서로 다른 단계로 인식하는가
- `4562 / 4797 / 3115`의 역할을 분리하는가
