# EA Part 3 Week 4 Day 1 Quickstart

기준일: 2026-04-24  
가정: 하루 `5시간`

이 파일은 Part 3 Week 4 `Day 1 accuracy` 전용 노트다.

## Day 1 핵심

- accuracy
- software reliance
- review of results
- miscalculations
- duplicate entries

## 오늘 목표

- filing process 문제를 계산문제가 아니라 `preparer responsibility` 문제로 본다
- software가 있어도 review obligation은 남는다는 감각을 만든다

## 오늘 문제 읽는 순서

- error source가 taxpayer data인가 preparer review failure인가
- software reliance가 excuse로 제시되는가
- accuracy failure인지 recordkeeping issue인지 구분되는가

## 오늘 꼭 외울 것

- software use does not eliminate the need to review the return
- accuracy questions are usually about process discipline, not hard tax computation

## 오늘 할 일

- accuracy 문제 `20~30문제`
- 아래 체크리스트를 손으로 다시 쓰기:
  - data entry review
  - duplicate entry check
  - obvious mismatch review
  - final return reasonableness review
