## Day 3 미니퀴즈

### 문제

1. Which split should you make first in most dependent questions?
   - A. HOH vs QSS
   - B. QC vs QR
   - C. W-2 vs 1099
   - D. Basis vs FMV

2. Which test is commonly associated with a qualifying child?
   - A. Residency
   - B. Estate tax
   - C. E&P
   - D. EFIN

3. Which statement is generally correct?
   - A. Gross-income test is the same as support test
   - B. Support and gross-income tests should be kept separate
   - C. Every dependent must be a qualifying child
   - D. Tie-breaker rules never matter

4. Which fact pattern is a common trap?
   - A. Full-time student
   - B. Corporate E&P
   - C. Form 3115
   - D. OIC

5. Which question should you ask early?
   - A. Who actually provided support?
   - B. Is this Form 706?
   - C. Is this depreciation recapture?
   - D. Is this a levy appeal?

6. Which statement is best?
   - A. QC and QR can be solved by the same exact test
   - B. Dependent questions are mostly about structured eligibility tests
   - C. Only age matters
   - D. Only residency matters
