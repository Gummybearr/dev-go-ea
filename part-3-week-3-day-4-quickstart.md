### 문제

1. Which distinction is fundamental in appeals questions?
   - A. 30-day letter vs 90-day letter
   - B. W-2 vs 1099-INT
   - C. basis vs FMV
   - D. QBI vs NOL

2. Which statement is generally best?
   - A. Timeliness of the request can control the taxpayer's rights
   - B. Timeliness rarely matters
   - C. Appeals only exists for collection
   - D. Appeals only exists for exam changes after payment

3. Which question should you ask early?
   - A. What exactly is being appealed?
   - B. Is this Section 351?
   - C. Is this NIIT?
   - D. Is this kiddie tax?

4. Which statement is generally correct?
   - A. Collection appeals and exam appeals should not always be treated as the same track
   - B. All appeals use the same document and same deadline
   - C. Appeals is unrelated to procedure
   - D. Form names do not matter

5. Which topic often appears with appeals?
   - A. conference / settlement function
   - B. depreciation class life
   - C. shareholder basis
   - D. farm income averaging

6. Which statement is best?
   - A. Appeals questions are usually about procedural posture, not memorizing buzzwords alone
   - B. Procedural posture is low value
   - C. Only substantive tax law matters
   - D. Deadlines never change outcomes
