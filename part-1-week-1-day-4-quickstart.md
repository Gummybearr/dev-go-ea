## Day 4 핵심

- filing requirement
- dependent filing rules
- standard deduction
- additional standard deduction
- due date
- extension

## 오늘 목표

- 누가 return을 내야 하는지를 `status + age + income type`으로 본다
- extension과 payment due date를 같은 것으로 보지 않는다

## 오늘 문제 읽는 순서

- taxpayer status와 age가 무엇인가
- dependent인가
- filing requirement를 만들 income type이 있는가
- extension인지 payment issue인지

## 오늘 꼭 외울 것

- extension of time to file is generally not an extension of time to pay
- dependent filing rules can be separate and stricter
- additional standard deduction commonly turns on age or blindness

## 오늘 할 일

- filing requirement / standard deduction 문제 `25~35문제`
- 아래 의사결정표를 손으로 다시 쓰기:
  - filing required?
  - dependent special rule?
  - standard deduction additions?

## Day 4 미니퀴즈

### 문제

1. Which statement is generally correct?
   - A. An extension to file usually extends the time to pay
   - B. An extension to file usually does not extend the time to pay
   - C. Due dates never matter
   - D. Dependents never need to file

2. Which factor commonly affects filing requirement?
   - A. Filing status
   - B. E&P
   - C. AAA
   - D. CAF

3. Which statement is generally correct?
   - A. Dependents may have their own filing requirement rules
   - B. Dependents always follow the same rule as nondependents
   - C. Standard deduction never changes
   - D. Age is irrelevant

4. Which issue can create an additional standard deduction?
   - A. Age 65 or older
   - B. Capital loss carryover only
   - C. FBAR filing
   - D. S election

5. Which question should you ask early?
   - A. Is this a filing obligation question or a payment-timing question?
   - B. Is this Form 1065?
   - C. Is this Section 351?
   - D. Is this OIC?

6. Which statement is best?
   - A. Day 4 is about deciding whether and how the taxpayer must file
   - B. Day 4 is mostly about partnership basis
   - C. Day 4 is mostly about appeals
   - D. Day 4 is mostly about foreign accounts
