## Day 6 미니퀴즈

### 문제

1. What is the main goal of a weak-area repair day?
   - A. Fix the most repeated mistakes first
   - B. Learn only brand-new topics
   - C. Ignore prior errors
   - D. Focus only on filing status

2. Which is a sensible repair target for Part 2?
   - A. Capitalization vs repair
   - B. HOH
   - C. Child tax credit
   - D. FBAR for individuals

3. Which statement is generally correct?
   - A. Repeated form confusion should be explicitly repaired
   - B. Old mistakes should not be revisited
   - C. Weak-area review is unrelated to Part 2 performance
   - D. Specialized returns should be skipped

4. Which cluster is a common Part 2 weak area?
   - A. Depreciation cluster
   - B. Filing status cluster
   - C. Individual education credits
   - D. FBAR only

5. Which question helps identify a real weak area?
   - A. What do I keep mixing up?
   - B. What is the weather?
   - C. What is the capital of France?
   - D. Which HOH rule applies?

6. Which statement is best?
   - A. Weak-area repair should target repeated error patterns, not random review
   - B. All topics should be treated equally every day
   - C. Form confusion does not matter
   - D. Basis mistakes are too minor to review
