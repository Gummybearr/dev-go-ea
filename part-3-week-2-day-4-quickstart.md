## Day 4 미니퀴즈

### 문제

1. Which statement is generally correct?
   - A. Authoritative vs non-authoritative source distinctions can matter in Part 3
   - B. IRS publications always carry the same weight as regulations
   - C. Supporting documents are irrelevant
   - D. Prior returns never matter

2. Which question should you ask early?
   - A. What is the strongest relevant authority here?
   - B. Is this HOH?
   - C. Is this FBAR?
   - D. Is this section 351?

3. Which cluster belongs together?
   - A. IRC / regs / rulings / case law / forms / publications / IRM
   - B. QBI / basis / E&P
   - C. 1023 / 990 / 4835
   - D. 4562 / 4797 / 3115

4. Which issue is part of supporting documentation?
   - A. Prior and subsequent tax returns
   - B. Child tax credit worksheet only
   - C. Schedule A medical logs only
   - D. HOH checklist

5. Which statement is best?
   - A. Representation questions can turn on both facts documentation and authority support
   - B. Source hierarchy never matters
   - C. Publications are always enough regardless of issue
   - D. IRM is the same as a regulation

6. Which is a legal-authority question rather than a collection-remedy question?
   - A. Whether a regulation is stronger authority than a publication
   - B. Whether to request CDP
   - C. Whether to submit Form 12153
   - D. Whether a levy is pending
