## Day 6 미니퀴즈

### 문제

1. What is the main purpose of Day 6?
   - A. Repair repeated weak areas
   - B. Learn a brand-new major domain
   - C. Skip review and rest
   - D. Memorize random acronyms

2. Which note format is most useful?
   - A. wrong trigger -> correct trigger
   - B. long narrative with no pattern
   - C. only question numbers
   - D. only scores

3. Which statement is generally correct?
   - A. Repeated confusion should be compressed into a few trigger rules
   - B. Every missed question needs a long essay
   - C. Review should ignore form selection
   - D. Weak-area repair is optional

4. Which issue is high value to repair?
   - A. form confusion
   - B. font preference
   - C. note color
   - D. page layout only

5. Which statement is best?
   - A. Day 6 should reduce repeated mistakes, not expand scope
   - B. Day 6 should broaden scope as much as possible
   - C. Day 6 should avoid mixed practice
   - D. Day 6 should ignore timing issues

6. Which statement is best?
   - A. A short repair note is better than a vague long note
   - B. Longer notes are always better
   - C. Trigger-based review does not help
   - D. Form names are unimportant
