# EA Part 2 시험 직전 1페이지 압축노트

기준일: 2026-04-24

이 문서는 Part 2 직전에 `빠르게 훑는 용도`다.  
핵심은 `entity -> tax level -> basis -> K-1 -> form` 순서를 자동화하는 것이다.

## 1. 문제 읽는 기본 순서

- entity가 무엇인가
- 누가 세금을 내는가
- 어떤 return을 쓰는가
- K-1이 나오는가
- basis / E&P / AAA / capital account 중 뭐가 중요한가
- deduction인가 capitalization인가
- 어떤 special taxpayer/form인가

## 2. Entity 레이더

- `sole prop` -> `Schedule C`
- `partnership` -> `1065 / K-1`
- `C corp` -> `1120`
- `S corp` -> `1120S / K-1`
- `trust / estate` -> `1041 / K-1`
- `exempt org` -> `990 series`

핵심:

- partnership, S corp는 `flow-through`
- C corp는 `entity-level tax`
- LLC는 `법형태`보다 `tax classification`을 먼저 본다

## 3. Partnership 압축

- `1065 -> K-1 -> partner`
- separately stated item
- guaranteed payment
- contribution of property
- liability assumption
- partner basis
- self-employment income 가능성

함정:

- guaranteed payment와 distributive share 혼동
- basis 계산에서 liabilities 누락
- K-1 item character 무시

## 4. C Corporation 압축

- `1120`
- entity-level tax
- E&P
- dividends / distributions
- redemption / liquidation
- accumulated earnings tax
- estimated tax

함정:

- distribution을 무조건 dividend로 처리
- E&P를 안 보고 distribution 문제 풂
- redemption과 liquidation 혼동

## 5. S Corporation 압축

- `1120S`
- `2553`
- qualification rules
- separately stated items
- shareholder basis / loan basis
- distributions
- losses in excess of basis

함정:

- 자격요건 누락
- shareholder basis 안 보고 loss/distribution 문제 풂
- 1120S와 2553 역할 혼동

## 6. Business Income / COGS / Expenses 압축

먼저 구분:

- gross receipts
- COGS
- other income
- deductible expense
- capitalized amount

핵심:

- accrual이면 earned but uncollected income 가능
- inventory / UNICAP
- ordinary / necessary / reasonable
- travel / meals / gifts / vehicle / insurance / taxes
- employment tax
- home office

## 7. Depreciation Cluster 압축

- `4562`
- section 179
- bonus depreciation
- regular depreciation
- amortization
- start-up / organizational cost
- recapture

함정:

- placed in service 무시
- 179와 bonus를 같은 것으로 처리
- recapture를 disposition 문제와 분리해서 봄

## 8. Assets / Capitalization / Accounting Methods 압축

- business property sale -> `4797`
- capitalization vs repair
- like-kind exchange
- converted property
- accounting methods
- `3115`
- books vs tax
- M-1 / M-2 / M-3

핵심:

- asset 문제는 `basis -> character -> form`
- capitalization vs repair는 고빈도
- accounting method change면 `3115` 가능성

## 9. Advising the Business Taxpayer 압축

- entity selection
- worker classification
- 1099 reporting
- Form 8300
- deposits / employment tax obligations
- accountable plan / mileage log
- commingling
- timing of income/expenses

함정:

- entity choice를 세율 하나로만 판단
- employee vs contractor를 대충 넘김
- 기록유지와 reporting obligations를 따로 안 봄

## 10. Specialized Returns 압축

- `1041` -> trust/estate
- `990 series` -> exempt org filing
- `1023 / 1024` -> exempt status application
- `4835` -> farm rental

키워드:

- DNI
- corpus vs income
- UBTI
- SEP / SIMPLE
- prohibited transaction
- Schedule J / Schedule SE
- real estate professional
- passive loss limitation

## 11. 폼 레이더

- `1065` -> partnership return
- `K-1` -> pass-through items
- `1120` -> C corp
- `1120S` -> S corp
- `2553` -> S election
- `4562` -> depreciation cluster
- `4797` -> sale of business property
- `3115` -> accounting method change
- `1041` -> trust/estate income tax
- `990 series` -> exempt org annual filing
- `1023 / 1024` -> exempt application
- `4835` -> farm rental

## 12. 시험장에서 마지막으로 볼 체크 15개

- entity를 먼저 봤는가
- taxed where를 먼저 정했는가
- K-1 여부를 확인했는가
- partnership basis / shareholder basis를 놓치지 않았는가
- E&P가 필요한 문제인가
- gross receipts와 COGS를 구분했는가
- deduction이 아니라 capitalization인지 검토했는가
- `4562` / `4797` / `3115`를 떠올렸는가
- worker classification이 붙는 문제인가
- 1099 / reporting obligation 문제인가
- trust/estate면 DNI를 떠올렸는가
- exempt org면 UBTI와 filing requirement를 떠올렸는가
- rental이면 passive loss limitation을 떠올렸는가
- farmer면 Schedule J / SE / 4835를 떠올렸는가
- form number를 보고 주제가 바로 떠오르는가

## 13. 시험 직전 암기 문장

- `Entity first, tax level second, form third.`
- `Partnership and S corp questions usually hide a basis issue.`
- `C corp distributions need an E&P mindset.`
- `Separate COGS from ordinary deductions.`
- `Depreciation questions start with placed in service and basis.`
- `Business asset sales point to Form 4797.`
- `Accounting method change points to Form 3115.`
- `Trust and estate questions point to DNI and Form 1041.`
