## Day 2 미니퀴즈

### 문제

1. Which issue should be screened early in a representation matter?
   - A. Potential criminal aspects
   - B. Schedule A
   - C. QBI
   - D. E&P

2. Which statement is generally correct?
   - A. Competence, expertise, and time-to-handle can matter in representation decisions
   - B. Conflict of interest is irrelevant in representation
   - C. Transcripts are a late-stage-only tool
   - D. Case-building is only about math

3. Which question should you ask early?
   - A. Can I competently handle this issue?
   - B. Is this HOH?
   - C. Is this 3115?
   - D. Is this 4835?

4. Which cluster belongs together?
   - A. Issue identification / criminal aspects / competence / conflict / transcripts
   - B. QBI / basis / E&P
   - C. DNI / corpus / 1041
   - D. 8948 / 8879 / 8453

5. Which statement is best?
   - A. Representation-case building begins before the practitioner starts arguing the merits
   - B. Conflict checks can be skipped
   - C. Transcripts are irrelevant
   - D. Criminal-aspect signals do not matter

6. Which issue is a representation-before-IRS issue rather than a filing-process issue?
   - A. Transcript use
   - B. Rejected e-file return resolution
   - C. Form 8453 mailing
   - D. EFIN revocation
